mysql异常1449

mysql异常1449

问题：CDbCommand 无法执行 SQL 语句: SQLSTATE[HY000]: General error: 1449 The user specified as a definer (‘root’@’%’) does not exist. The SQL statement executed was: select getChildUserByID(256)

解决办法：mysql执行命令grant all privileges on . to root@”%” identified by “.”;对不同主机IP授权命令
若无立即生效参考：注意授权后必须FLUSH PRIVILEGES;否则无法立即生效。
